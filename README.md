# Тестовое задание SaaSoft

## Настройка проекта

### Установка зависимостей

```sh
npm install
```

### Сборка в режиме разработки

```sh
npm run dev
```

### Проверка типов и сборка для продакшена

```sh
npm run build
```

### Только сборка для продакшена

```sh
npm run build-only
```

### Только проверка типов

```sh
npm run type-check
```

### Линтинг кода

```sh
npm run lint
```

### Форматирование кода

```sh
npm run format
```
